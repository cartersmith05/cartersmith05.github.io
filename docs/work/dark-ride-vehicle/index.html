Dark Ride Vehicle

🗓️ 2023

A dark ride vehicle designed in my introductory engineering class. It was part of a group project, and I was responsible for the Fusion 360 modeling and 3D printing.

Overview

This project was designed to create an exciting theme park ride that takes riders through a Wild West story.

Process

One of my group members completed the technical drawings for the ride vehicle, which I then translated into Fusion 360. The vehicle sits on a ride platform and features a moving arm bar that allows riders to enter and exit.

Lessons Learned

This project was my first in-depth experience with Fusion 360 and was extremely helpful in learning the software's interface. In the future, I plan to improve my use of components when creating different vehicle parts, so that returning to make updates or improvements is a more seamless process.
